About SentiScan

NIET, Greater Noida · Under supervision of Mr. Ankit Sharma

🧠 What It Does

SentiScan analyzes any text and instantly tells you if it's positive, negative or neutral — AND identifies the specific emotion behind it (joy, anger, sadness, etc.) using advanced AI.

⚙️ How It Works

Text goes through NLP preprocessing → TF-IDF feature extraction → ML classification (Naive Bayes, Logistic Regression, LSTM, BERT) → LLM emotion detection → Dashboard visualization.

🎯 Why It Matters

Businesses analyze millions of reviews, tweets and messages daily. Doing this manually is impossible. SentiScan automates this with 95%+ accuracy in real-time.

🚀 Key Innovation

Unlike basic tools that only give Positive/Negative, SentiScan detects 6 granular emotions using LLM APIs — a first in unified sentiment + emotion dashboards.

🛠️ Technology Stack

React.js Python Flask/FastAPI BERT LSTM Naive Bayes Logistic Regression NLTK spaCy TF-IDF Scikit-learn TensorFlow OpenAI API Chart.js MongoDB
